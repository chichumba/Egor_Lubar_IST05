# 🎬 OMDb API Client - Jupyter Notebook Edition

Улучшенный Python скрипт для поиска информации о фильмах через OMDb API с полной обработкой ошибок, валидацией и безопасностью.

## 🚀 Быстрый старт

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Создать конфиг файл
cp .env.example .env

# 3. Добавить API-ключ в .env
# Получить здесь: https://www.omdbapi.com/apikey.aspx

# 4. Открыть Jupyter
jupyter notebook

# 5. Запустить rest_api_demo_improved.ipynb
```


## 🐛 Проблемы и решения

### OMDB_API_KEY not found
Создайте `.env` файл и добавьте свой ключ

### Movie not found
Проверьте правильность названия на английском языке

### Connection timeout
Проверьте интернет соединение
